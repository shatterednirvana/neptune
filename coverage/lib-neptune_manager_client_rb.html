62   # A helper method to make SOAP calls for us. This method is mainly here to
 
63   # reduce code duplication: all SOAP calls expect a certain timeout and can
 
64   # tolerate certain exceptions, so we consolidate this code into this method.
 
65   # Here, the caller specifies the timeout for the SOAP call (or NO_TIMEOUT
 
66   # if an infinite timeout is required) as well as whether the call should
 
67   # be retried in the face of exceptions. Exceptions can occur if the machine
 
68   # is not yet running or is too busy to handle the request, so these exceptions
 
69   # are automatically retried regardless of the retry value. Typically
 
70   # callers set this to false to catch 'Connection Refused' exceptions or
 
71   # the like. Finally, the caller must provide a block of
 
72   # code that indicates the SOAP call to make: this is really all that differs
 
73   # between the calling methods. The result of the block is returned to the
 
74   # caller. 
75   def make_call(time, retry_on_except)
 
76     begin
 
77       Timeout::timeout(time) {
 
78         yield if block_given?
 
79       }
 
80     rescue Errno::ECONNREFUSED
 
81       if retry_on_except
 
82         retry
 
83       else
 
84         raise NeptuneManagerException.new("Connection was refused. Is the NeptuneManager running?")
 
85       end
 
86     rescue OpenSSL::SSL::SSLError, NotImplementedError, Timeout::Error
 
87       retry
 
88     rescue Exception => except
 
89       if retry_on_except
 
90         retry
 
91       else
 
92         raise NeptuneManagerException.new("We saw an unexpected error of the type #{except.class} talking to #{@ip} with the following message:\n#{except}.")
 
93       end
 
94     end
 
95   end
